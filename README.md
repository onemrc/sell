# sell
基于SpringBoot的点餐系统

跟着慕课网上面的一个教程做的，用来入门SpringBoot，后续功能慢慢完善

实现功能点：
 - 前后端对接
 - 买家端类目
 - 买家端商品
 - 买家端订单
 - 卖家端订单
 - 卖家端通用功能和上下架
 - 卖家端新增商品和类目


主要技术点：
- SpringBoot
- Hibernate
